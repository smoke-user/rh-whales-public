# 🐋 RobinhoodChain Whale Coins — public dashboard

Live whale analytics for Robinhood Chain (chain id 4663): what the top-860
native-balance wallets actually BUY (payment-verified on-chain), hold, sell,
or merely receive as airdrops — plus a 💎 Gem Score and a token analysis
widget (liquidity, holder concentration, risk flags).

**Data files** (`data/*.json`) are pushed automatically by the crawler; every
push triggers a Vercel redeploy, so the site always shows the latest run.

- On-chain data: [Blockscout](https://robinhoodchain.blockscout.com)
- Prices/liquidity: [DexScreener](https://dexscreener.com)
- Static site, no backend, no tracking.

Not financial advice. DYOR.
